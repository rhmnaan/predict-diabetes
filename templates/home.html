Prediksi Risiko Diabetes

Selamat datang di platform prediksi risiko diabetes berbasis machine learning.

Tujuan Web Ini

Situs ini dirancang untuk memberikan prediksi risiko diabetes berdasarkan data pengguna menggunakan model Machine Learning.

  • Edukasi Dini: Memberikan perkiraan awal mengenai potensi risiko diabetes.
  • Kesadaran: Meningkatkan kesadaran terhadap faktor kesehatan yang berkontribusi pada diabetes.
  • Demonstrasi: Menampilkan bagaimana machine learning digunakan untuk prediksi kesehatan.

Model ini dilatih dengan dataset Pima Indian Diabetes menggunakan algoritma Random Forest untuk memprediksi status "Diabetes" atau "Tidak Diabetes" beserta probabilitasnya.

*Penafian: Alat ini hanya untuk keperluan edukasi dan demonstrasi. Tidak menggantikan diagnosis atau saran medis profesional.*
